🤖 بوت إدارة ماين كرافت عبر الديسكورد | Minecraft Discord Bot

بوت ديسكورد احترافي ومتكامل مبرمج ومطور باستخدام مكتبة Discord.js (v14) ومكتبة Mineflayer لإدارة وتشغيل حسابات ماين كرافت (بوتات الـ AFK) مباشرة من داخل سيرفر الديسكورد الخاص بك عبر قنوات إدارية مخصصة ولوحة تحكم تفاعلية متكاملة.

🌟 المميزات البرمجية الحالية

🛠️ واجهة تثبيت تفاعلية ومنبثقة (Modals Setup): تثبيت ذكي وسريع للنظام بأكمله عبر أمر السلاش /setup والذي يظهر نافذة منبثقة تفاعلية لتعبئة إعدادات السيرفر دون أي تعديل يدوي في ملف الأكواد.

🛡️ نظام حماية ذكي ضد الطرد (Anti-Kick Shield): يتفادى البوت مشكلة الطرد الشائعة invalid_player_movement من خلال تصفير حالته الحركية تماماً (clearControlStates) فور دخوله اللعبة، ويكتفي بتحديث زاوية الكاميرا لإعلام حماية السيرفر بنشاطه دون التسبب بأي خطأ فيزيائي ناتج عن الحركة.

🎮 لوحة تحكم تفاعلية متطورة: تشغيل وإيقاف البوت بضغطة زر مع دعم اختيار إصدار ماين كرافت المفضل بدقة (من 1.16.5 إلى 1.20.4 أو الكشف التلقائي) لضمان استقرار البوت في بيئة اللعب.

💬 ربط شات مزدوج مباشر (Two-Way Chat Bridge):

إرسال شات ماين كرافت والرسائل مباشرة إلى قناة السجلات في الديسكورد لحظياً.

إرسال الرسائل والأوامر من قناة الشات المخصصة أو من قناة السجلات (كـ Console) مباشرة للعبة وتأكيد وصولها بوضع التفاعلات التلقائية (✅) أو (❌).

🔒 تأمين القنوات تلقائياً: يقوم البوت بإنشاء تصنيف (Category) وقنوات مخفية بالكامل عن الجميع، وتظهر فقط للرتبة الإدارية المسموح لها والتي يتم تحديدها في البداية لضمان الخصوصية والأمان.

📁 قاعدة بيانات ذكية ومحلية: حفظ جميع الإعدادات محلياً في ملف database.json لضمان استمرارية الاتصال وعدم ضياع البيانات عند إعادة تشغيل البوت.

📂 هيكلة ملفات المشروع (Directory Structure)

index.js — الملف البرمجي الأساسي للبوت والمحرك.

database.json — قاعدة البيانات المحلية لحفظ البيانات (يتم إنشاؤها تلقائياً).

package.json — ملف الاعتمادات وتعريف المشروع وتثبيت الحزم.

README.md — دليل الاستخدام والتثبيت (هذا الملف).

📋 المتطلبات الأساسية (Prerequisites)

Node.js: الإصدار 16.11.0 أو أحدث.

Discord Bot Token: توكن البوت الخاص بك من صفحة المطورين لـ Discord.

Discord Gateway Intents: يجب تفعيل الخيارات التالية في لوحة تحكم مطوري الديسكورد (Developer Portal):

GUILDS

GUILD_MESSAGES

MESSAGE_CONTENT (مهم جداً لقراءة شات الديسكورد وتمريره للعبة).

🚀 طريقة التثبيت والتشغيل

1. تحميل ملفات المشروع وتثبيت الحزم

npm install discord.js mineflayer


2. إعداد متغيرات البيئة (Environment Setup)

قم بإنشاء متغير بيئة بالنظام باسم DISCORD_TOKEN وضع فيه توكن البوت الخاص بك، أو قم بإنشائه داخل ملف بيئة .env إذا كنت تستخدم حزمة مخصصة:

DISCORD_TOKEN=your_discord_bot_token_here


3. تشغيل البوت

قم بتشغيل ملف البوت الأساسي عبر الأمر:

node index.js


🛠️ دليل الاستخدام والإعداد بالديسكورد

توجه إلى سيرفر الديسكورد واكتب أمر السلاش التالي (متاح فقط للإداريين):

/setup [allowed_role: اختر الرتبة المسؤولة] [server_type: نوع السيرفر (مكرك أو أصلي)]


ستظهر لك نافذة منبثقة (Modal) تطلب منك إدخال تفاصيل البوت كالتالي:

🤖 اسم البوت (اللاعب): اسم حساب ماين كرافت (مثال: AFK_Warrior).

🌐 عنوان السيرفر (IP): عنوان الـ IP الخاص بسيرفر ماين كرافت.

🔌 بورت السيرفر: منفذ الاتصال (اتركه فارغاً ليعتمد تلقائياً البورت الافتراضي 25565).

💬 اسم قناة الشات: اكتب الاسم الذي ترغب به لقناة الشات المباشرة مع اللعبة.

عند الإرسال، سيقوم البوت تلقائياً بإنشاء تصنيف جديد (Category) مخفي وصلاحيات مخصصة ويحتوي على القنوات التالية:

🎮 | لوحة-التحكم: تحتوي على أزرار تشغيل وإيقاف البوت.

⚙️ | الإعدادات: تتيح لك الاطلاع على بيانات البوت وتحديثها في أي وقت بضغط زر.

📜 | سجلات-البوت: تعرض سجلات اللعبة (شات ماين كرافت، الدخول، الخروج، وحالات الطرد) وتعمل كـ Console لإرسال الأوامر والرسائل مباشرة للعبة.

💬 | [اسم القناة المختار]: قناة الشات المباشر للتفاعل الفوري مع اللاعبين.

💡 نصائح هامة للوقاية من الطرد

تحديد الإصدار يدوياً: عند الضغط على "🟢 تشغيل البوت"، ننصح بشدة باختيار إصدار السيرفر بشكل دقيق من القائمة المنسدلة وتجنب خيار "تلقائي (Auto Detect)". تحديد الإصدار يدوياً يضمن مطابقة تامة لمحرك الفيزياء.

البوت صامت وراكد: تم برمجة البوت ليبقى واقفاً تماماً في مكان رسبرته دون أي تحرك مشي أو قفز، وهو الوضع الأكثر أماناً على الإطلاق لتفادي الحمايات والـ Anti-Cheat.
